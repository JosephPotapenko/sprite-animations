# SpriteForge 256

SpriteForge 256 is a browser-based sprite animation tool that runs well in GitHub Codespaces and on a normal CPU-only machine. Open the repo in Codespaces, wait for dependency install, then run:

```bash
uvicorn app.main:app --host 0.0.0.0 --port 8000 --reload
```

The actual app lives in [spriteforge_256_full/spriteforge_256_full](spriteforge_256_full/spriteforge_256_full). For full setup notes and API details, see the nested README in that folder.
